# Structural Design of a G+2 Residential Building

## Overview

This project involved the analysis, design, and verification of a G+2 residential building using both Reinforced Concrete (RC) and Structural Steel framing systems. The work included preliminary sizing, load calculations, ETABS modelling, manual design verification, reinforcement detailing, Demand-Capacity Ratio (DCR) assessment, and comparison of RC and steel structural performance under gravity loading conditions.

## Project Objectives

* Develop 3D analytical models of RC and Steel building frames.
* Perform structural analysis under dead and live loads.
* Design RCC beams, columns, and steel members according to Indian Standards.
* Verify manual calculations using ETABS outputs.
* Evaluate member adequacy through DCR checks and serviceability requirements.

## Building Configuration

* Building Type: G+2 Residential Building
* Plan Dimensions: 9 m × 9 m
* Storey Height: 5.5 m
* Structural Systems:

  * RC Frame (IS 456:2000)
  * Steel Frame (IS 800:2007)
* Analysis Type: Gravity Load Analysis (DL + LL)

## Methodology

### 1. Preliminary Structural Design

* Developed column grid, beam layout, and slab configuration.
* Estimated preliminary member sizes using span-depth ratios and tributary area methods.
* Calculated dead loads, live loads, and governing load combinations.

### 2. ETABS Modelling & Analysis

* Created 3D ETABS models for RC and Steel frames.
* Applied gravity load combinations.
* Generated BMDs, SFDs, deflection profiles, and support reactions.
* Evaluated member forces and structural behaviour.

### 3. Manual Design Verification

* Designed RCC beams using Limit State Method.
* Designed RCC columns based on axial load requirements.
* Designed steel beams and columns according to IS 800:2007.
* Compared manual calculations with ETABS-generated design results.

### 4. Design Validation

* Performed flexural, shear, and serviceability checks.
* Evaluated Demand-Capacity Ratios (DCR) for critical members.
* Optimized reinforcement and member sections based on ETABS feedback.
* Corrected and verified critical roof beam design through DCR analysis.

## Key Results

### Reinforced Concrete Frame

* Final beam section: 230 mm × 450 mm
* Final column section: 300 mm × 450 mm
* All beam and column DCR values remained below 1.0 after design revision.
* Reinforcement detailing prepared for beams and columns.

### Steel Frame

* Designed steel beams and columns using IS 800:2007 provisions.
* Verified member adequacy using DCR checks.
* Compared steel frame behaviour with RC frame performance.
